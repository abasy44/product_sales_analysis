
# 📊 Product Sales Analysis  
# 🧾 تحليل مبيعات منتج معين  

## Overview  
**This is a simple data analysis project built with Python and Plotly.**  
It demonstrates how to clean, analyze, and visualize sales data for a single product.  

**الغرض من المشروع:**  
- استكشاف بيانات مبيعات منتج معين.  
- حساب إجمالي ومتوسط المبيعات.  
- عرض الأداء من خلال مخططات تفاعلية.  

---

## 🔧 Technologies Used  
**التقنيات المستخدمة:**  
- Python 3  
- Plotly Express  
- CSV file handling (معالجة ملفات CSV)  

---

## 📁 Project Structure  
**هيكل المشروع:**

product-sales-analysis/ ├── main.py               # Main script for analysis and visualization ├── sales_data.csv        # Sample sales data (date, units sold) └── README.md             # This documentation file

## 📈 Sample Output  
**الناتج المتوقع:**  
An interactive line chart showing product sales trends over time.  
مخطط خطي تفاعلي يوضح اتجاهات مبيعات المنتج مع مرور الوقت.
